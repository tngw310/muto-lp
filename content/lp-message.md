# LP メッセージ戦略（武藤雄大）

## LPの役割定義

### 主目的：支援者向け情報ハブ

政治家のLPは「広く有権者に認知させる」よりも、**すでに関心を持っている人を支援者に転換し、支援者が活動しやすい情報を提供する**役割が強い。

```
[認知層] → [関心層] → [支援層] → [活動層]
             ↑                    ↑
           LPの入口            LPの出口
```

### なぜ支援者向けか

1. **流入経路**: SNS、紹介、街頭で見かけた → すでに名前を知っている人が来る
2. **広告費用**: 政治広告は制限が多く、大量リーチが難しい
3. **選挙活動**: 支援者が「広報素材」を持って動く必要がある
4. **組織化**: 後援会員・ボランティアの獲得が直接的な選挙力になる

## ターゲットペルソナ

### Primary: 潜在的支援者（関心層→支援層）

> 「この人いいかも」と思い始めた人を「応援しよう」に変える

| 属性 | 詳細 |
|-----|------|
| 想定人物 | 佐藤健太（35歳）的な子育て通勤者世帯 |
| 流入経路 | SNSで「手取り」関連を見た、駅前で見かけた、国民民主党で検索 |
| 心理状態 | 「国民民主党の千葉6区って誰？」「どんな人なんだろう」 |
| 求める情報 | 人柄、政策（特に手取り・子育て）、信頼できるか |
| 行動目標 | SNSフォロー、イベント参加検討 |

### Secondary: 活動的支援者（支援層→活動層）

> 「応援したい」人を「一緒に動く」人に変える

| 属性 | 詳細 |
|-----|------|
| 想定人物 | すでに街頭活動を見た、イベントに参加した |
| 心理状態 | 「何か手伝いたい」「周りにも広めたい」 |
| 求める情報 | 広報素材、イベント情報、ボランティア方法 |
| 行動目標 | 素材DL・拡散、後援会入会、ボランティア参加 |

### Tertiary: メディア・関係者

> 取材・講演依頼時の公式情報源

| 属性 | 詳細 |
|-----|------|
| 流入経路 | 取材準備、記事執筆 |
| 求める情報 | 正確な経歴、高解像度写真、連絡先 |
| 行動目標 | 素材DL、問い合わせ |

## 支援者のJTBD（Jobs To Be Done）

### Functional Job（機能的なJob）

> 「政治家に何をしてほしいか」ではなく「自分の人生で何を達成したいか」

| Job | 完了状態 | 政治との接点 |
|-----|---------|-------------|
| 家計に余裕を作りたい | 毎月5万円貯金できる | 103万→178万円の壁解消、さらなる手取り増 |
| 子育ての負担を減らしたい | 出費のタイミングで支援が届く | 児童手当500万円、選択制、申請レス |
| 将来の見通しを持ちたい | 老後・教育費の計画が立つ | 社会保険料の見直し、安心できる制度 |

### Emotional Job（感情的なJob）

| Job | 完了状態 |
|-----|---------|
| 「頑張りが報われている」と感じたい | 給料が上がったら手取りも増える実感 |
| 「自分の声が届いている」と感じたい | 政治家が自分の言葉で語っている |
| 「この人を応援してよかった」と思いたい | 実際に政策が実現した |

### Social Job（社会的なJob）

| Job | 完了状態 |
|-----|---------|
| 「ちゃんとした親」として見られたい | 子どもの進路で「お金がない」と言わずに済む |
| 「政治に関わっている」と言えるようにしたい | 応援している候補がいると言える |

## UVP（Unique Value Proposition）

### ポジショニング・ステートメント

> **松戸で働き、子どもを育てる現役世代**のために、
> 武藤雄大は**「現場と政治をつなぎ、制度を機能させる政治家」**です。
>
> 立憲・自民と違い、**「手取りを増やす」を本気で実現した国民民主党の一員として**、
> **あなたの声を届け、家計に余裕が生まれる社会**を実現します。

### UVPの3つの柱

| 柱 | 内容 | 競合との差別化 |
|----|------|---------------|
| 1. 手取りを増やす | 103万→178万円の実績。さらなる減税を推進 | 立憲は増税イメージ、自民は消極的 |
| 2. 声を届ける | 「見えない声を見えるようにする」記者経験 | 安藤氏は県議実績、武藤は現場発 |
| 3. 制度を機能させる | 政策を作って終わりにしない | 永田町の論理ではなく現場で検証 |

> 差別化で勝つ前に、最低限の土俵に立つ必要がある

| PoP項目 | 有権者が期待する水準 | 現状 | LPでの対応 |
|--------|---------------------|------|-----------|
| 地元との接点 | 「松戸の人」と認識される | △ | 千葉出身・松戸での活動を強調、地元写真多用 |
| 政策の網羅性 | 主要分野をカバー | ◯ | 3本柱（経済・子育て・防災）を明示 |
| 信頼性・経歴 | 「ちゃんとした人」 | ◯ | NHK記者経験を前面に |
| 露出・認知度 | 名前と顔を知ってもらう | × | FVで顔写真＋名前を大きく、動画活用 |

## RTB（Reason To Believe）

> なぜ武藤がこれを実現できると信じられるか

| 主張 | RTB（根拠） |
|-----|------------|
| 「届かない声を届ける」 | 記者として災害現場・地方政治を取材し、条例・制度を動かした実績 |
| 「手取りを増やす」 | 国民民主党が103万→178万円を実現。党として動かした証拠 |
| 「子育て当事者として」 | 0歳児の父。当事者としてリアルな痛みを知っている |
| 「制度を機能させる」 | 「政策を作って終わりにしない」という姿勢をキャリアで一貫 |

## 認識変容設計

### Primary: 潜在的支援者

| Before | After |
|--------|-------|
| 「国民民主党の候補って誰？」 | 「武藤さんって人がいるんだ」 |
| 「政治家ってみんな同じでしょ」 | 「記者出身で、現場を見てきた人なんだ」 |
| 「手取りを増やすって本当？」 | 「103万→178万円、実際にやったんだ」 |
| 「自分には関係ない」 | 「同じ子育て世代で、自分ごとで語ってる」 |
| 「応援って何すればいいの」 | 「まずSNSフォローしてみよう」 |

**変容のキー**:
- **当事者性**: 「0歳児の父」としてリアルな痛みを語る
- **実績**: 103万→178万円は「言うだけ」ではないRTB
- **低いハードル**: まずはSNSフォローから

### Secondary: 活動的支援者

| Before | After |
|--------|-------|
| 「応援したいけど何すれば」 | 「このチラシを配ればいいんだ」 |
| 「素材を探すのが面倒」 | 「ここに全部揃ってる」 |
| 「ボランティアってハードル高い」 | 「まずイベントに顔出してみよう」 |

**変容のキー**:
- 広報素材をワンストップで提供
- ボランティア・後援会の案内を具体的に
- 「一緒に声を届ける仲間」という帰属意識

## メッセージ設計

### ワンフレーズ（全ペルソナ共通）

> **「届かない声を、届ける。」**

- identity（原体験）と一貫
- 「声を届ける」という役割が明確
- 支援者にとっても「自分の声を届けてくれる人」と共感できる

### キーメッセージ（3本）

| # | メッセージ | 伝えたいこと | 対象文脈 |
|---|-----------|-------------|---------|
| 1 | **「働いた分が、ちゃんと暮らしに残る社会を取り戻す」** | 手取りを増やす。103万円→178万円の実績 | 経済 |
| 2 | **「子育ての"今しんどい"に応える」** | 0歳児の父として当事者の声を届ける | 子育て |
| 3 | **「届かない声を、届ける政治家になる」** | 記者経験を活かし、現場と政治をつなぐ | 政治姿勢 |

### ペルソナ別CTA

| ペルソナ | メッセージ | CTA |
|---------|----------|-----|
| 潜在的支援者 | 「あなたの声を、一緒に届けませんか」 | SNSフォロー、イベント参加 |
| 活動的支援者 | 「あなたの力で、もっと届ける」 | 素材DL、後援会入会、ボランティア |
| メディア・関係者 | 「正確な情報を、ここから」 | プロフィールDL、問い合わせ |

## トーン＆マナー

### 推奨

- **親しみやすさ**: 政治家っぽくない、等身大の言葉
- **誠実さ**: 大言壮語しない、「まだ道半ば」も言う
- **当事者性**: 「私も0歳児の父として」
- **行動喚起**: 「一緒にやろう」という姿勢

### 避けるべき（Non-message準拠）

| 言わないこと | 理由 |
|-------------|------|
| 「全世代のために」 | 主顧客（子育て通勤者世帯）を明確にする |
| 政権批判中心 | 批判より「何をするか」を語る |
| 「私が変える」という英雄的表現 | 主役は有権者。「あなたの声を届ける」が軸 |
| 憲法・外交の踏み込んだ主張 | 主顧客の「今の痛み」に直結しない |
