# LP 構成案（武藤雄大）

## 設計方針

### ペルソナ優先度に基づく構成

```
[上部] 潜在的支援者向け → 共感獲得、信頼構築、認知獲得
[中部] 詳細情報 → 政策、経歴、RTB
[下部] 活動的支援者向け → CTA、素材、組織化
```

### 閲覧パターン想定

1. **ライトユーザー**: FV → 挨拶 → プロフィール概要 → 離脱 or SNSフォロー
2. **検討ユーザー**: 上記 + 政策 → RTB確認 → SNSフォロー or イベント参加検討
3. **支援者**: 全体スクロール → 素材DL、後援会入会、イベント確認

## セクション構成

### 1. ヘッダー（固定）

| 要素 | 内容 | 備考 |
|-----|------|------|
| ロゴ/名前 | 武藤雄大 | クリックでTOPへ |
| ナビゲーション | 政策/プロフィール/応援する | ページ内スクロール |
| SNSアイコン | X/Instagram/YouTube | 新規タブで開く |
| CTAボタン | 「応援する」 | 目立つ色、下部CTAへ |

---

### 2. ファーストビュー（FV）

**目的**: 3秒で「誰が」「何を目指しているか」を伝える。**認知度が低いため、顔と名前を大きく**

| 要素 | 内容 | 備考 |
|-----|------|------|
| メインビジュアル | 武藤の写真（親しみやすい表情、松戸での活動写真） | 顔を覚えてもらう |
| 名前 | **武藤雄大**（大きく） | 読み仮名も |
| 肩書 | 国民民主党 千葉6区（松戸市） | 党と地域を明示 |
| キャッチコピー | **「届かない声を、届ける。」** | ワンフレーズ |
| サブコピー | 0歳児の父。記者出身。現場から政治へ。 | 当事者性＋経歴 |
| 動画ボタン（任意） | 自己紹介動画へのリンク | 「武藤を知る」 |

**PoP対応**: 認知度が低い → FVで顔＋名前を最大限に押し出す

---

### 3. 自己紹介動画セクション

**目的**: 動画で人柄を伝える（テキストより訴求力高い、RTBにもなる）

| 要素 | 内容 |
|-----|------|
| YouTube埋め込み | 1〜2分の自己紹介動画 |
| 見出し | 「1分で分かる武藤雄大」 |
| 他動画リンク | YouTubeチャンネルへ |

---

### 4. 挨拶・理念セクション

**目的**: なぜ政治を志すのか、価値観を伝えて共感獲得

| 要素 | 内容 |
|-----|------|
| 見出し | 「なぜ、政治家を目指すのか」 |
| 本文 | 政治理念（300〜400字） |

**構成（政見放送準拠）**:
1. **原体験**: 福島での「届かない声」との出会い
2. **記者としての経験**: 現場と政治の距離の遠さ
3. **決意**: 「伝える側」から「決める側」へ
4. **当事者性**: 0歳児の父として

**キーフレーズ**:
> 「現場と政治の距離が、あまりにも遠い。だから私は『伝える側』ではなく『決める側』に行くと決めました。」

---

### 5. 政策セクション

**目的**: 何を実現しようとしているか具体的に示す（UVPの3本柱）

| 要素 | 内容 |
|-----|------|
| 見出し | 「武藤雄大の政策」 |
| 構成 | 3本柱をカード形式で |

#### 3本柱の構成

| # | 政策 | キーフレーズ | RTB |
|---|------|-------------|-----|
| 1 | **経済・手取り** | 「働いた分が、ちゃんと暮らしに残る」 | 103万→178万円実現 |
| 2 | **子育て** | 「子育ての"今しんどい"に応える」 | 0歳児の父、500万円児童手当 |
| 3 | **防災** | 「待つ支援から、届ける支援へ」 | 記者として災害現場を取材 |

#### 各政策カード

| 要素 | 内容 |
|-----|------|
| アイコン | 経済→円マーク、子育て→家族、防災→盾 |
| タイトル | 政策名 |
| サブタイトル | キーフレーズ |
| 概要 | 50〜100字 |
| 数字（RTB） | 「103万→178万円」「500万円」など |

**詳細リンク**: 政策PDF、または別ページ（任意）

---

### 6. プロフィールセクション

**目的**: 信頼性を担保（PoP: 「ちゃんとした人」）、当事者性を伝える

#### 6-1. 基本情報

| 項目 | 内容 |
|-----|------|
| 顔写真 | 正面、笑顔 |
| 氏名・ふりがな | 武藤雄大（むとう ゆうだい） |
| 生年月日 | ◯年◯月◯日生まれ |
| 出身地 | **千葉県習志野市**（千葉出身を強調） |
| 家族 | 妻、0歳の子ども（当事者性） |
| 所属 | 国民民主党 千葉6区（松戸市）総支部長 |

#### 6-2. 経歴（RTBとしての設計）

| 時期 | 経験 | RTBとしての意味 |
|-----|------|----------------|
| 学生時代 | 早稲田大学政経学部、米国・北京留学、福島での被災地支援 | 原体験、国際的視野 |
| 記者時代 | NHK記者（山形→宮城→東京）、災害・行政・地方政治取材 | **「現場を見る力」「制度を動かす実績」** |
| POTETO Media | 政治分野ディレクター、政治家の広報サポート | 政治とメディアの接点 |
| 現在 | 国民民主党 千葉6区総支部長、0歳児の父 | 当事者として活動中 |

**強調ポイント**:
- NHK記者 → 「ちゃんとした人」のRTB
- 福島での経験 → 原体験としてのRTB
- 0歳児の父 → 子育て政策の当事者性

#### 6-3. 原体験エピソード（任意）

政見放送の導入部分を短縮して掲載。

> 学生時代、東日本大震災後の福島に通い、原発事故の影響を受けた高校生たちと向き合った。「支援される側だから」「言っても意味がない」——自分の本音を飲み込んでしまう若者たちがそこにいた。届かない声が、社会にはあまりにも多い。

### 8. 応援する/CTAセクション

**目的**: 支援行動への転換（最重要セクション）

#### 段階的CTA設計

| 応援レベル | アクション | ハードル | 設置場所 |
|-----------|----------|---------|---------|
| ライト | SNSフォロー | 低 | 上部・中部・下部 |
| ミドル | イベント参加、メルマガ登録 | 中 | 中部・下部 |
| ヘビー | 後援会入会、ボランティア | 高 | 下部のみ |

#### 構成

1. **見出し**: 「一緒に、届かない声を届けませんか」
2. **応援方法の説明**:
   - **まずはフォローから**: SNSアイコン（X/Instagram/YouTube）
   - **イベントに参加する**: 直近スケジュール or 「SNSで告知」
   - **もっと応援する**: 後援会入会フォーム、ボランティア案内
3. **メッセージ**: 「あなたの一票、あなたの声が、政治を動かします」
