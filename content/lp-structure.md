# LP 構成案（武藤雄大）

## 設計方針

### ペルソナ優先度に基づく構成

```
[上部] 潜在的支援者向け → 共感獲得、信頼構築、認知獲得
[中部] 詳細情報 → 政策、経歴、RTB
[下部] 活動的支援者向け → CTA、素材、組織化
```

### 閲覧パターン想定

1. **ライトユーザー**: FV → 挨拶 → プロフィール概要 → 離脱 or SNSフォロー
2. **検討ユーザー**: 上記 + 政策 → RTB確認 → SNSフォロー or イベント参加検討
3. **支援者**: 全体スクロール → 素材DL、後援会入会、イベント確認

### ベンチマーク参照

| 参考サイト | 取り入れる要素 |
|-----------|---------------|
| 小泉進次郎 | 署名画像、政策の体系的整理、SNS埋め込み、ダウンロード素材 |
| 古井康介 | エピソード形式、動画活用、応援の声、参加型CTA、「60秒で分かる」 |

### 4. 挨拶・理念セクション

**目的**: なぜ政治を志すのか、価値観を伝えて共感獲得

#### 構成要素

| 要素 | 内容 |
|-----|------|
| 見出し | 「なぜ、政治家を目指すのか」 |
| 本文 | 政治理念（300〜400字） |
| 署名画像（任意） | 手書き風サイン |

#### 本文構成（政見放送準拠）

1. **原体験**: 福島での「届かない声」との出会い
2. **記者としての経験**: 現場と政治の距離の遠さ
3. **決意**: 「伝える側」から「決める側」へ
4. **当事者性**: 0歳児の父として

#### 本文案（約350字）

> 学生時代、東日本大震災後の福島に通い、原発事故の影響を受けた高校生たちと向き合いました。「支援される側だから」「言っても意味がない」——自分の本音を飲み込んでしまう若者たちがそこにはたくさんいました。
>
> 届かない声が、社会にはあまりにも多い。
>
> NHK記者として災害現場や地方政治を取材する中で、現場と政治の距離があまりにも遠いことを痛感しました。声は途中で消え、制度はあるのに、現場では機能しない。
>
> だから私は、「伝える側」ではなく、「決める側」に行くと決めました。
>
> いま私は、0歳の子どもを育てる父親です。この千葉6区も、子育て世代、働く世代が本当に多い地域です。
>
> **「届かない声を、届ける。」**
>
> これが私の使命です。

#### ベンチマーク

- **小泉**: 約400字の挨拶文＋署名画像。選挙敗北への反省→保守政党としての姿勢→決意
- **古井**: 約500字。子どもたちの未来→日本の現状への危機感→政策づくりへの決意
